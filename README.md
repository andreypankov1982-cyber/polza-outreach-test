# Polza Agency — тестовое задание «Вайбкодер-аутричер»

Репозиторий со скриптом для тестового задания.

## Что внутри

- `personalize_companies.py` — Python-скрипт для генерации персонализации и QA-отчёта по базе.
- `companies.csv` — входная база компаний.
- `vibe_stack.txt` — стек и инструменты, которые использовались.

## Как запустить

```bash
python personalize_companies.py --input companies.csv --output companies_with_personalization.csv --report validation_report.csv --check-urls
```

## Что делает скрипт

1. Читает CSV с компаниями.
2. Проверяет наличие обязательных колонок.
3. Заполняет колонку `Персонализация`, если она пустая.
4. Проверяет формат email.
5. Опционально проверяет доступность сайтов компаний.
6. Создаёт файл результата и отдельный QA-отчёт.

## Результат

- `companies_with_personalization.csv` — база с персонализацией.
- `validation_report.csv` — отчёт по email, ссылкам и проблемным строкам.

## Комментарий

Email взяты из публичных источников, указанных в таблице. SMTP/deliverability-проверка почтовых ящиков не выполнялась.
