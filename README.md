# Polza Agency — тестовое задание «Вайбкодер-аутричер»

Репозиторий со скриптом для тестового задания на позицию «Вайбкодер-аутричер».

## Что внутри

- `polza_personalize_companies.py` — Python-скрипт для генерации персонализации и базовой QA-проверки базы.
- `polza_companies.csv` — входная база компаний.
- `polza_vibe_stack.txt` — стек и инструменты, которые использовались.

Полная Google Таблица содержит два листа:

- `База` — список B2B-компаний, сайты, email, источники, краткие факты и персонализация.
- `Письма` — цепочка из 3 писем для холодного outreach.

## Как запустить

```bash
python polza_personalize_companies.py
```

Скрипт читает файл `polza_companies.csv` и создаёт файл с результатом.

## Что делает скрипт

1. Читает CSV с компаниями.
2. Проверяет наличие обязательных колонок.
3. Заполняет колонку `Персонализация`, если она пустая.
4. Проверяет формат email.
5. Создаёт итоговый CSV-файл.

## Результат

После запуска скрипта формируются 4 файла:

- `companies_with_personalization.csv` — стандартный CSV с обновлённой базой и персонализацией.
- `validation_report.csv` — стандартный CSV-отчёт проверки базы.
- `companies_with_personalization_excel.csv` — версия базы с разделителем `;`, чтобы файл удобно открывался в русской версии Excel по колонкам.
- `validation_report_excel.csv` — версия отчёта с разделителем `;`, чтобы файл удобно открывался в русской версии Excel по колонкам.

Файлы без `_excel` оставлены как стандартный CSV-формат для скриптов и GitHub.

Файлы с `_excel` добавлены отдельно для удобной ручной проверки результата в Excel.

Для сдачи тестового используются:
